## [1.0.0] - 2026-01-13

### Added
- Django UI and REST API for infrastructure automation scripts
- AWS Inventory Export (multi-region, multi-resource XLSX output)
- VPC Migration Toolkit with Terraform bundle generation
- HA VPN Builder for AWS-to-GCP migration planning
- Classic VPN Builder with dependency mapping
- ECR to Artifact Registry migration workflow
- React frontend with credential management and artifact download
- CLI entry points via `python -m feature.<script>`
- Shared automation task registry for pluggable script integration
- AWS/GCP helper endpoints for VPC, subnet, and network discovery
